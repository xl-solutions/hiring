O BuscaGit foi feito usando HTML, CSS (Bootstrap) e JavaScript(Vanilla)

=> Execução

Para executá-lo, após a obtenção dos arquivos é necessário que o arquivo "index.html" seja aberto com um navegador da web.
Seu funcionamento consiste na inserção do nome de usuário que deseja ser buscado no espaço em evidência e após isso é necessário o clique no botão "Buscar" logo abaixo.
Feito isso, serão exibidos o nome do usuário, sua foto de perfil e seus repositórios, estes apresentando seus nomes, url de clonagem e um link para acesso a página original do repositório no GitHub.
