## Executar na seguinte ordem:
```
1º backend
```
```
2º frontend
```
***
- Após clonar o projeto, entrar na pasta ``backend`` e executar o comando:
```
yarn install
```
 - Repetir o processo para a pasta ``frontend``


### Para iniciar o backend
```
$ cd backend/
$ yarn start
```

### E por fim:
```
$ cd frontend/
$ yarn start
```
